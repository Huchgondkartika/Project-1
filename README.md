# Library Management Project

This is a Library Management System project developed in Java.  
The project helps to manage books, students, and book issue/return records in a library.  
It is created for learning Java programming concepts and implementing object-oriented programming.


## Project Description
The Library Management System allows users to:
- Add new books to the library
- View available books
- Search books by title/author
- Register students
- Issue and return books
- Track issued book records

This project demonstrates:
- Classes and Objects
- Constructors and Methods
- File handling / Database connectivity (optional)
- Collections framework (optional)
- Basic Java I/O
